# VIBE FIGHT

ブラウザで遊べる 1対1 の 2D 格闘ゲーム。`index.html` を開くだけ（ビルド・インストール不要）。

🎮 **遊ぶ:** https://wish-hub-hackathon.vercel.app/

## 特長

- **CPU戦**（5段階の難易度）と、ルームコードでつなぐ**オンライン対戦**
- グラフィックもサウンドも**すべてコードで生成**（外部アセット 0・完全オフライン）
- 戦うほど**ポイント＆ランク**が貯まる（記録はブラウザに自動保存）

## 操作

| 操作 | キー |
| --- | --- |
| 移動 | `← →` |
| 攻撃 | `A`〜`Z` 連打（押すたびに技がループ） |
| ガード | 相手の攻撃中に逆方向へ歩く |
| 決定 / リスタート | `Enter` |

スーパーゲージが満タンになると、次の攻撃で**必殺技「波動・烈」**が発動します。

## ルール

体力を削り合い、先に 0 にした方がラウンド勝利。**3本勝負（2本先取）・制限時間60秒**。

## 起動（ローカル）

```bash
# index.html をダブルクリックでもOK。サーバー経由なら:
python3 -m http.server 8000   # → http://localhost:8000
```
※ 音は最初のキー入力後に鳴り始めます。

## 技術

単一 HTML（バニラ JS）／ Canvas 2D ／ Web Audio API ／ オンラインは PeerJS（P2P）。
`main` への push で Vercel に自動デプロイ。
