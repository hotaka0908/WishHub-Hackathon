# NEON FIST — ネオン 2D 格闘ゲーム

PCのブラウザで動く、ストリートファイター風の1対1 格闘ゲーム。
**CPU戦**と、ルームコードでつなぐ**オンライン対戦**が遊べます。
**外部ファイル・ビルド不要** — `index.html` をブラウザで開くだけ。
グラフィックは Canvas 2D の関節アニメ、サウンドは Web Audio でリアルタイム合成（音声ファイル一切なし）。

🎮 **今すぐ遊ぶ:** https://neon-fist.vercel.app/

---

## 起動方法

- 一番簡単: `index.html` をダブルクリックしてブラウザ（Chrome 推奨）で開く
- ローカルサーバー経由:
  ```bash
  cd WishHub-Hackathon
  python3 -m http.server 8000
  # ブラウザで http://localhost:8000 を開く
  ```
- ※ 音はブラウザの仕様上、**最初のキー入力後**に鳴り始めます。

## モード選択（タイトル画面）

`↑ ↓` で選択し、`Enter` または数字キーで決定します。

| モード | 内容 |
| --- | --- |
| **CPU戦** | コンピューターと対戦 |
| **オンライン対戦** | ルームコードを共有して 1対1（P2P接続） |

## 操作方法

| 操作 | CPU戦 | オンライン対戦 |
| --- | --- | --- |
| 移動 | `← →` | `← →` または `A` / `D` |
| 攻撃 | 文字キー（`F` / `Space` など） | `F` / `Space` |
| ガード | 相手の攻撃中に相手と逆方向へ歩く | 同左 |
| スタート / リスタート | `Enter` | （オンラインは画面の「もう一度 / 退出」） |

### 攻撃のループ
攻撃キーを押すたびに、次の技が順番に出てループします（どのキーかは不問）:

1. **弱パンチ**
2. **強パンチ**
3. **ジャンプキック**
4. **ダイナミックキック**
5. **大車輪パンチ**（両手を振り回す大振り）
→ 1 に戻る

### 必殺技「波動・烈」
攻撃を**当てる／受ける**と上部の黄色い**スーパーゲージ**が溜まり、**満タン（MAX）で必殺技が自動発動**。前方へ飛び道具を放ち、貫通＆大ダメージ＋シネマ演出が入ります。

## ルール

- 体力を削り合い、先に 0 にした方がラウンド勝利
- **3本勝負（2本先取）** ／ 制限時間 **60秒**（時間切れは体力が多い方の勝ち）
- ガードは相手の攻撃と逆方向に歩くと成立（削りダメージのみ受ける）

## 演出・サウンド

- **打撃音はリアルタイム合成**：モーダル合成（高Qバンドパスの共鳴）による“当たった芯”＋重低音＋破裂バースト。コンボで音程が上昇し、空間リバーブで余韻がつく
- **技ごとに音を出し分け**（パンチ／キック／スピン）
- **対戦BGM**：疾走するバトルリフ（キック＋スネア＋ベース＋リード）
- **演出**：ヒットストップ／画面シェイク／フラッシュ／シネマズーム／衝撃波／スピードライン／ダメージ数字・コンボ表示
- **試合の節目の音**：対戦START スティンガー、必殺技の大爆発音、KO の決着音、勝利ファンファーレ／敗北ジングル

## 技術メモ

- **単一 HTML**（HTML + CSS + バニラ JS）。ライブラリ・ビルド不要
- **Canvas 2D**：背景パララックス、関節キャラのアニメ、各種エフェクトを描画
- **Web Audio API**：効果音・BGM をすべてコードで合成（アセット 0）。マスターにコンプ／リミッタ、打撃用のリバーブバスを構成
- **オンライン対戦**：PeerJS（ルームコードで P2P 接続、ロックステップで入力同期）
- **デプロイ**：GitHub → Vercel 自動デプロイ（`main` への push で本番反映）

## デプロイ

GitHub リポジトリが Vercel と連携済みで、`main` ブランチへ push すると自動で本番デプロイされます。

```bash
./update.sh "変更内容のメモ"   # まとめてコミット＆push（自動デプロイ）
```
