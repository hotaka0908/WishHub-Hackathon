# NEON FIST — 対CPU 2D格闘ゲーム

PCのブラウザで動く、ストリートファイター風の1対1 格闘ゲーム（対コンピューター）。
**外部ファイル・ビルド不要**。`index.html` をブラウザで開くだけで遊べます。
グラフィックはCanvas 2Dの関節アニメ、サウンドはWeb Audioで合成（音声ファイル不要）。

## 起動方法
- 一番簡単: `index.html` をダブルクリックしてブラウザ（Chrome推奨）で開く。
- もしくはローカルサーバー経由:
  ```bash
  cd "WishHub-Hackathon"
  python3 -m http.server 8000
  # ブラウザで http://localhost:8000 を開く
  ```
- ※音はブラウザの仕様上、最初のキー入力後に鳴り始めます。

## 操作方法
| キー | 動作 |
| --- | --- |
| ← / → | 左右に移動 |
| A〜Z（どれでもOK） | 攻撃。**押すたびに5種類が順番にループ** |
| 同じキーを素早く2回 | 特別技（強化攻撃） |
| 同じキーを長押し | 必殺技（スーパーゲージを50消費・飛び道具） |
| Enter | スタート / リスタート |
| ガード | 相手の攻撃中に、相手と逆方向へ歩く（削りダメージのみ） |

### 攻撃のループ（A〜Zを連打）
どの文字を押したかは関係なく、押すたびに次の技が出てループします:

1. **弱パンチ**
2. **強パンチ**
3. **ジャンプキック**
4. **ダイナミックキック**
5. **大車輪パンチ**（両手を振り回す大振り）
→ 1に戻る

画面左上の「NEXT ▶」に次に出る技が表示されます。特別技（2回押し）・必殺技（長押し）は、その時点のループ位置に対応した強化技が出ます。

## ルール
- 体力ゲージを削り合い、先に0にした方がラウンド勝利。
- **3本勝負（2本先取）**。制限時間60秒。時間切れは体力が多い方の勝ち。
- 攻撃を当てる/受けると上部の黄色い**スーパーゲージ**が溜まり、満タン付近で必殺技が撃てます。

## 技術メモ
- 単一HTML（HTML + CSS + バニラJS）。Canvas 2D で背景パララックス・関節キャラ・エフェクト（火花/リング/画面シェイク/ヒットストップ/スロー演出）を描画。
- Web Audio API で効果音・簡易BGMを合成。
- 完全オフライン、CDN・アセット不要。
